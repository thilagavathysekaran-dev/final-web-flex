This project is a simple college event registration webpage designed using HTML and CSS.
It provides a minimal user interface for students to enter their details and register for a college event.

Features:
Two square input boxes for user details
Flexbox used to align elements side by side
Clean and centered layout
Beginner-friendly design

Technologies Used:
HTML5
CSS(Flexbox)

Files Included:
index.html – Structure of the registration page
style.css – Styling and Flexbox layout
